# Python Security Scripts

A collection of Python scripts built while learning Python fundamentals, 
with a focus on cybersecurity concepts.

## Scripts

### sets/security_access_checker.py
Simulates a cloud security access control system.
- Checks if users have unauthorized permissions
- Identifies admin access
- Finds shared permissions across users

## Concepts Practiced
- Python Sets
- Set operations (issubset, union, intersection, difference)
- If/Else logic

## Goal
These scripts are part of my journey towards becoming a Cloud Security Engineer.
# Python Security Scripts

A collection of Python scripts built while learning Python fundamentals, 
with a focus on cybersecurity concepts.

## Scripts

### sets/security_access_checker.py
Simulates a cloud security access control system.
- Checks if users have unauthorized permissions
- Identifies admin access
- Finds shared permissions across users

### dictionaries/cloud_user_management.py
Simulates a cloud user management system.
- Displays all users and their details
- Checks active/inactive status
- Adds new users
- Activates and deactivates users
- Removes users from the system
- Displays remaining users and their roles

## Concepts Practiced
- Python Sets
- Set operations (issubset, union, intersection, difference)
- Python Dictionaries
- Nested dictionaries
- Dictionary methods (update, pop, items, keys, values)
- If/Else logic

## Goal
These scripts are part of my journey towards becoming a Cloud Security Engineer.
